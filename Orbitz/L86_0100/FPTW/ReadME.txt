Based on Windows 64 bit environment.


*for Entry ME and Full ME*

-------------------Vpro Off--------------------------------
Set Vpro off with AT off
1. Execute VoffNoAT.bat
2. Execute CLSMNF.bat

*Only for Full ME*
-------------------Vpro On with UMA platform--------------------------------
Set Vpro on with AT off
1. Execute VonNoAT.bat
2. Execute CLSMNF.bat


*Only for Full ME*
-------------------Vpro On with Worksation DreamColor-DIS platform only---------------
Set Vpro on with AT off with No KVM
1. Execute DISNoAT.bat
2. Execute CLSMNF.bat
